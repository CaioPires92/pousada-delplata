# Use só um fluxo por deploy:

```Deploy manual completo (recomendado no seu caso ZIP):
npm run hostinger:full:allphotos
Se quiser ZIP menor e aceitar só fotos referenciadas no código:
npm run hostinger:full:lean
Se já está tudo buildado e quer só gerar ZIP rápido:
npm run hostinger:zip:allphotos:fast ou npm run hostinger:zip:lean:fast```
